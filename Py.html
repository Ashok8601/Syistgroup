Step 1: Tkinter ka Introduction

Tkinter ek built-in GUI library hai Python me. Iska use karke aap windows, buttons, input boxes etc. create kar sakte ho.

Step 2: Basic Calculator Layout Design


from tkinter import *

root = Tk()
root.title("Simple Calculator")

entry = Entry(root, width=35, borderwidth=5)
entry.grid(row=0, column=0, columnspan=3, padx=10, pady=10)

Yeh code ek basic window aur ek input box create karega.

Step 3: Button Functions Create Karna


def button_click(number):
    current = entry.get()
    entry.delete(0, END)
    entry.insert(0, current + str(number))

Yeh function numbers ko screen par dikhata hai jab user button dabata hai.

Step 4: Mathematical Function Add Karna


def button_add():
    first_number = entry.get()
    global f_num
    global math
    math = "addition"
    f_num = int(first_number)
    entry.delete(0, END)

def button_equal():
    second_number = entry.get()
    entry.delete(0, END)
    if math == "addition":
        entry.insert(0, f_num + int(second_number))

Step 5: Buttons Create Karna


button_1 = Button(root, text="1", padx=40, pady=20, command=lambda: button_click(1))
button_1.grid(row=3, column=0)
